# reti-ghost-sdk

Fast Reti SDK that utilizes [ghost cotracts](https://github.com/d13co/ghostkit) to batch fetch data.

**Status: Experimental**

## Development Installation

**For production instructions and SDK usage, see `projects/sdk/README.md`**

For SDK development, run at project root:

```
algokit bootstrap all
```

## Development Build

Run at project root:

```
algokit project run build
```
